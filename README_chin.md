## 命令录  
#### (black|white) [A-S] [1-19]  
把 黑/白 棋放到对应的坐标  
#### remove [A-S] [1-19]  
清除一个棋子  
#### clear  
清空棋盘  
#### file  
档案管理系统   
###### save [name]  
存储当前游戏 
###### load [name]
载入游戏  
###### delete [name]  
删除游戏  
###### list  
列出所有游戏
